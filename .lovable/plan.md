
# Propuesta de Rediseño: Nuevo Sitio Institucional Capittal

Basándome en el análisis de las referencias (Ethos Venture y Miura Partners) y el estado actual del diseño, propongo una evolución completa que eleve la presencia digital de Capittal al nivel de las mejores firmas de inversión europeas.

```text
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │

│                                                                         │
```

## Mejoras Adicionales Propuestas

### Animaciones
- Scroll-triggered animations para cada sección
- Counter animation para stats (ej: €0 → €902M)
- Parallax sutil en imágenes de fondo
- Transiciones suaves entre secciones

### Responsive
- Slider táctil en móvil
- Cards apiladas verticalmente
- Footer simplificado
- Hero con altura adaptativa

### Performance
- Lazy loading de imágenes
- Preload de siguiente slide
- Optimización de animaciones (GPU)
- Critical CSS inline
